The project serves as the backend for coursework, providing a simplified implementation of the Piazza social platform API. 
It offers functionality including user registration and login, posting, upvoting, downvoting, commenting, and management of expiring health posts.

This project includes:
 
Source code (src/): API service implemented with Express + Node.js

• Data model (models/): MongoDB / Mongoose

• Routes (routes/): Auth, Posts, Interactions

• Middleware (middleware/): JWT user authentication

• Dockerfile: Used to build container images

• .dockerignore: Optimizes Docker builds

• Postman Collection: Used for testing TC1–TC20
